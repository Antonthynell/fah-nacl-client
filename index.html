Folding@home NaCl Client

Download .zip Download .tar.gz View on GitHub

This project contains the source code for the Open-Source frontend of the Folding@home NaCl Web Client. This client requires the Google Chrome browser. You can run the client either from the Chrome Web Store or by going directly to http://folding.stanford.edu/nacl/.

Email joseph@cauldrondevelopment.com for more info.
